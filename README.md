A quick start for using gnuplot for in-terminal plotting

Change Space:

	me -space 8,6

Change Size:

	me -size 200,125
 
Change Index:

	me -I '(a)'

Change Caption:

	me -ic '{s//a}'

Change Key:

	me -K p

 	me -K s

Change Axis:

	me -xl '{s//l}_{//R}'

 	me -yl '{//dI/dV}'

Help:

	me -help
